[BAGIAN 1: FILOSOFI DESAIN & TECH STACK]

4.1. Visi "Jarvis" Command Center
Frontend Sentinel WaterSense dirancang untuk memberikan pengalaman "High-Tech 
Operations Room". Penggunaan tema gelap dengan aksen Cyan dan Emerald memberikan 
kesan teknologi militer yang presisi.

4.2. Arsitektur Teknologi:
- **Tailwind CSS**: Untuk desain utilitas tinggi dan performa rendering cepat.
- **Leaflet.js**: Engine pemetaan geospasial untuk pelacakan node.
- **Chart.js**: Engine visualisasi data untuk telemetri air.
- **Blade Templating**: Rendering sisi server yang dioptimalkan untuk SEO.

[BAGIAN 2: BEDAH ANTARMUKA PER-HALAMAN]

1. PUBLIC MONITORING PORTAL (`user_dashboard.blade.php`)
Halaman utama bagi publik untuk memantau ketinggian air secara langsung.
- **Visual Fokus**: Fokus pada angka TMA raksasa di bagian tengah (Cockpit View).
- **Interaktivitas**: Update otomatis angka dan grafik sparkline setiap 5 detik 
  tanpa refresh halaman.

2. IT COMMAND CENTER (`it_dashboard.blade.php`)
Pusat kendali operasional bagi tim IT Admin.
- **System Health Monitor**: Widget di bagian atas yang memantau uptime server 
  dan status database.
- **Live Stream Log**: Panel scrolling yang menampilkan data telemetri mentah 
  yang baru masuk untuk keperluan debugging cepat.

3. DEVICE ASSET HQ (`devices/index.blade.php`)
Halaman manajemen inventaris perangkat sensor.
- **Orbital Descent Animation**: Efek zoom sinematik dari luar angkasa menuju 
  lokasi Karawang saat halaman dibuka.
- **Snippet Animasi**:
```javascript
map.flyTo([-6.3227, 107.3376], 12, { duration: 5, easeLinearity: 0.25 });
```

4. DEVICE DETAILED SPEC (`devices/show.blade.php`)
Halaman bedah teknis untuk setiap alat secara individual.
- **Hardware Profile**: Menampilkan detail spesifikasi (Wemos D1 Mini, JSN-SR04T).
- **Interactive Geotagging**: Peta statis dengan marker presisi dan koordinat 
  derajat desimal (DD).

5. ANALYTICS INTELLIGENCE HUB (`it/analytics/index.blade.php`)
Pusat analisis data historis jangka panjang.
- **Dual-Axis Visualization**: Grafik sumbu ganda yang membandingkan level air 
  dengan ambang batas bahaya (Siaga 1-3).
- **Export Engine**: Antarmuka untuk mengekspor data ribuan baris ke format CSV.

6. PERSONEL CONTROL CENTER (`it/users/index.blade.php`)
Manajemen akses dan keamanan akun.
- **Role Badge System**: Identifikasi visual cepat antara IT_ADMIN dan ADMIN.
- **Secure Modal**: Form pendaftaran user baru dengan validasi keamanan ketat.

7. TACTICAL AUDIT TRAIL (`it/users/history.blade.php`)
Halaman pelacakan seluruh aktivitas yang dilakukan oleh personil di sistem.
- **Action Timeline**: Garis waktu vertikal yang mencatat setiap perubahan 
  konfigurasi alat atau pengaturan sistem.

[BAGIAN 8: FUTURISTIC AUTHENTICATION SUITE (LOGIN & REGISTER)]

4.8. Login Experience (`auth/login.blade.php`)
Pintu gerbang sistem ini dirancang untuk memberikan kesan pertama "High-End Security".

A. Atmosferik Background
Sistem menggunakan teknik "Moving Blobs" untuk latar belakang yang dinamis namun tetap ringan.
```html
<div class="absolute top-[-10%] left-[-10%] w-[50%] h-[50%] rounded-full bg-blue-400/20 blur-[120px] animate-pulse"></div>
```
Penjelasan: Penggunaan `blur-[120px]` menciptakan gradasi warna yang sangat halus (Atmospheric Depth) yang berdenyut pelan, memberikan kesan sistem yang "hidup".

B. Glass Branding Panel (Left Panel)
Hanya muncul di layar desktop, panel ini berfungsi sebagai identitas visual sistem.
- **Glass Effect**: Menggunakan `backdrop-blur-xl` dan `border-white/60`.
- **Branding Tags**: Menampilkan tag "Real-time Telemetry" dan "Safety Guaranteed" dengan desain badge yang modern.

C. Advanced Access Form (Right Panel)
- **Google SSO**: Tombol login cepat dengan efek `group-hover:scale-110` pada logo Google.
- **Form Focus Power-Up**: Input field yang berubah dari border standar ke `ring-2 ring-blue-500/20` saat diklik, memberikan feedback visual yang tegas.
- **Interactive Submit**: Tombol login menggunakan gradien dinamis:
```html
<button class="bg-gradient-to-r from-slate-900 to-slate-900 hover:from-blue-600 hover:to-blue-700 transition-all duration-500">
```

4.9. Registration & Onboarding Flow (`auth/register.blade.php`)
Sama seperti login, halaman register menggunakan layout simetris dengan fokus pada kemudahan pendaftaran personil baru.
- **Validation State**: Penggunaan warna merah pada border dan teks saat terjadi kesalahan input (Error Handling) menggunakan direktif `@error` Laravel.

[BAGIAN 9: SECURITY & PROFILE COMPLETION]

4.10. Complete Profile & Change Password
Dua halaman ini (`complete_profile.blade.php` & `change_password.blade.php`) adalah bagian dari alur keamanan tinggi.
- **Mandatory Logic**: User baru tidak bisa mengakses dashboard sebelum melengkapi profil dan mengubah password default.
- **UI Consistency**: Tetap menggunakan tema Glassmorphism agar transisi antar halaman tidak merusak pengalaman visual pengguna.

[BAGIAN 10: SYSTEM EMERGENCY LOCKDOWN (RED ALERT)]

4.11. Halaman Lockdown (`errors/503.blade.php`)
Ini adalah fitur keamanan paling krusial. Saat sistem dikunci (Lockdown), halaman ini akan menggantikan seluruh antarmuka aplikasi.
- **Visual Alert**: Animasi perisai berdenyut (`shield-pulse`) dan garis pemindaian (`scanline`) berwarna merah.
- **Snippet CSS**:
```css
@keyframes scanline {
    0% { transform: translateY(-100vh); }
    100% { transform: translateY(100vh); }
}
```
- **Fungsi**: Mencegah akses ke seluruh database dan fitur sistem selama situasi darurat atau pemeliharaan kritis.

[BAGIAN 3: SISTEM ANIMASI & MICRO-INTERACTIONS]

4.12. Staggered Reveal Logic
Sistem tidak memunculkan elemen secara kaku, melainkan menggunakan delay 
bertahap (0.2s per elemen) sehingga dashboard terasa "hidup" saat pertama kali 
dimuat.

4.13. Indicator Real-time
Penggunaan CSS `animate-ping` pada marker peta memberikan kepastian visual 
kepada operator bahwa perangkat di lapangan sedang aktif mengirimkan data.
